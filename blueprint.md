# **App Name**: Varnika Ledger

## Core Features:

- Purchase Logging: Record daily gas cylinder purchases (14.2kg, 19kg, 10kg, 5kg).
- Sales Logging: Log daily cylinder sales, differentiating between counter sales and sales by Upender, Veerababu, and Srinivas, specifying cylinder type (14.2kg, 19kg, 10kg, 5kg).
- Expense Logging: Record daily operational expenses including daily wages, diesel costs, electricity bills, chit payments, and custom expense entries.
- Dynamic Reporting: AI tool analyzes transaction history and detects potential cost savings
- Real-time Summary: Generate summaries in a sheet, dynamically showing current cylinders (14.2kg, 19kg, 10kg, 5kg) stock.
- Reporting and Exporting: Generate reports with daily, monthly, and yearly views, and facilitate exporting data in Excel sheet format.
- User Authentication: Secure user authentication system to protect financial data and prevent unauthorized access.

## Style Guidelines:

- Primary color: Dark teal (#367588), conveying trust and reliability.
- Background color: Light gray (#EAECEE), providing a neutral backdrop for data presentation.
- Accent color: Mustard yellow (#FFDA63), for highlighting important data points and interactive elements.
- Headline font: 'Space Grotesk', sans-serif. Body Font: 'Inter', sans-serif.
- Use a set of consistent, simple icons for navigation and representing different types of transactions and data.
- A clean, tabular layout for presenting ledger data, with clear categorization and easy-to-read formatting.
- Subtle transitions and animations when loading data or navigating between different views to enhance user experience.
